# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-11

### Initial Release

- AES-256-GCM authenticated encryption
- Argon2id key derivation (scrypt fallback)
- CLI commands: init, encrypt, decrypt, show, run, verify, check, rotate, export-vars
- TypeScript library API
- Comprehensive test coverage
- Security hardened (memory wiping, file permissions, no secret leaks)
